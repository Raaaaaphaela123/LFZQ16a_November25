## Java Wiederholungsquiz 

## Frage 1: Welche der folgenden `for`-Schleifen zählt von 1 bis 5?

- A) `for(int i = 1; i < 5; i++)`  
- B) `for(int i = 1; i <= 5; i++)`  - richtig!
- C) `for(int i = 1; i < 6; i--)`  
- D) `for(int i = 0; i <= 5; i--)`

## Frage 2: Was passiert bei folgender Schleife?

```java
int i = 0;
while(i < 3) {
    System.out.println(i);
}
```
- A) Es wird 0, 1, 2 ausgegeben.
- B) Es wird 0 unendlich oft ausgegeben. - richtig!
- C) Die Schleife läuft einmal und endet.
- D) Kompiliert nicht.
