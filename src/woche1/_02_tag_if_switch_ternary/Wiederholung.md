## Java Wiederholungsquiz 

# Frage 1: Welcher der folgenden Bezeichner ist gültig in Java?
- 2name
- _value ✓
- @count
- user-name

# Frage 2: Welche Regeln gelten für Java-Bezeichner?

- Darf mit einer Zahl beginnen 
- Darf Unterstriche enthalten ✓
- Darf Leerzeichen enthalten
- Groß- und Kleinschreibung ist relevant ✓

# Frage 3: Welche der folgenden Variablendeklarationen ist ungültig?

- int count = 0;
- double 1value = 3.5; ✓
- boolean isReady = true;
- char letter = 'A';

# Frage 4: Welcher Operator wird für die Modulodivision (Rest) verwendet?
- /
- % ✓
- *
- //

# Frage 5: Was ist das Ergebnis des folgenden Ausdrucks?
```Java
int x = 5;
int y = ++x * 2;
```
Lösung: y = 12

# Frage 6: Welche Aussagen zu logischen Operatoren stimmen?

- && führt einen logischen UND-Vergleich aus ✓
- || führt einen logischen ODER-Vergleich aus ✓
- ! negiert einen Wahrheitswert ✓
- & ist immer identisch zu &&

# Frage 7: Ist folgender Code gültig? Begründe.
```Java

int class = 10;
```
Lösung: Ungültig, "class" ist ein keyword.

# Frage 8: Was gibt folgender Code aus?
```Java

int a = 7;
int b = 3;
System.out.println(a / b);
System.out.println(a / (double) b);
```
Lösung: 2 und 2.3333
