# Java Quiz – ArrayList & OOP Basics

### 1. Wie fügt man ein Element in eine ArrayList ein?
- (a) `list.add(element);` ✓
- (b) `list.insert(element);`
- (c) `list.put(element);`
- (d) `list.push(element);`

### 2. Wie greift man auf das 2. Element einer ArrayList zu?
- (a) `list[1]`
- (b) `list.get(1)` ✓
- (c) `list.get(2)`
- (d) `list[2]`

---

### 3. Welche Aussage über ArrayLists ist richtig?
- (a) Die Größe ist festgelegt und kann nicht verändert werden

### 4. Wie entfernt man ein Element an Index 3 aus einer ArrayList?

- (b) `list.delete(3);`
- (c) `list.removeElement(3);`
- (d) `list.removeAt(3);`
